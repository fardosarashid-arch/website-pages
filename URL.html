Uniform Resource Locator (URL)

A URL (Uniform Resource Locator) is the address of a unique resource on the internet. It is one of the key mechanisms used by browsers to retrieve published resources, such as HTML pages, CSS documents, images, and so on..

Structure

  • Scheme (Protocol): Defines how the resource is accessed (e.g., https:// for secure web traffic, http:// for standard web, ftp:// for file transfers
  • Subdomain: A subdivision of the main domain, often used to organize site sections (e.g., blog.example.com or www).
  • Domain Name: The unique name of the website (e.g., example in example.com).
  • Top-Level Domain (TLD) / Domain Extension: The suffix (.com, .org, .net).
  • Port Number: Identifies the specific port on the server to connect to (usually hidden, e.g., 80 for HTTP or 443 for HTTPS).
  • Path: Defines the specific resource or page location within the server's file structure (e.g., /blog/article-name).

Example

https://www.example.com/page
