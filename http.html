Home

Introduction

HTTP is the protocol used to transfer web pages from a web server to a web browser.

Functions of HTTP

  1. Transfers web pages between client and server
  2. Uses request-response communication
  3. Supports multimedia content (text, images, video)
  4. Works on a stateless connection
  5. Uses URLs to locate resources

HTTP Request Methods

  • GET – retrieve data
  • POST – send data
  • PUT – update data
  • DELETE – remove data
