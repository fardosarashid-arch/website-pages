What is the Web?

The World Wide Web (WWW) is a system of interlinked web pages that are accessed through the internet using web browsers.

Main Components

  • Web Pages – documents written in HTML
  • Web Browsers – software used to access websites
  • Web Servers – computers that store and deliver web pages

How the Web Works

  1. User enters a URL
  2. Browser sends request to server
  3. Server processes the request
  4. Webpage is sent back and displayed
