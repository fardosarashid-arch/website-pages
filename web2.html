Overview

Web communication is the process by which data is exchanged between a client (browser) and a server.

Steps in Communication

  1. User enters a web address
  2. DNS translates the domain name
  3. Browser sends HTTP request
  4. Server processes request
  5. Server sends HTTP response
  6. Browser displays the webpage

Protocols Used

  • HTTP / HTTPS
  • TCP/IP
  • FTP
