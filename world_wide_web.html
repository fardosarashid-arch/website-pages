WORLD_WIDE_WEB

definition

The World Wide Web is a global system of interlinked digital documents and resources that are accessed over the Internet using web browsers. It uses technologies such as Hypertext Transfer Protocol (HTTP) to transfer data and Uniform Resource Locators (URLs) to identify and locate web pages. The Web allows users to navigate between pages through hyperlinks and supports various types of content, including text, images, videos, and interactive media.

key features

  • uses hyperlinksto connect to web page
  • supports multi media content(text, images, videos, audio)
  • works with web browsers such as chrome, firefox and microsoft Edge
  • Uses standard technologies like HTTP, HTML, and URLs

How it works

  1. A user enters a URL into a web browser.
  2. The browser sends a request to a web server using HTTP.
  3. The server processes the request and finds the required web page.
  4. The server sends the web page back to the browser.
  5. The browser displays the page for the user to view and interact with.
